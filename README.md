Zorvyn_Frontend Developer Assignment

Financial Dashboard – Angular 19

A responsive Financial Dashboard built using Angular 19, Bootstrap 4, and Chart.js.
This dashboard allows users to track transactions, view income/expense summary, and visualize data using charts.

Features
- Dashboard summary cards
- Income and expenses tracking
- Transactions table
- Role based UI (Admin / Viewer)
- Charts visualization
- Insights section
- Responsive design

Tech Stack
- Angular
- Bootstrap 4
- TypeScript
- Chart.js

How to run

1. npm install
2. ng serve
3. open http://localhost:4200

Role

Viewer: View only
Admin: Add/Edit transactions
